In this assignment we were required to translate 2D points into pixels with certain given color and depth.
After that, we were required to create triangles between the given points and paint the triangles with 2 different methodologies.
After that is done, a picture is produced.

Running either one of the demo matlab files (2) will produce a racoon from the given data in racoon_hw1
in a certain painting style (flat/gouraud).
